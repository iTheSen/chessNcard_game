# 纸牌程序需求核对清单

## 一、规则总结（按需求文档）

1. 初始有一张底牌（手牌顶部），主牌区卡牌与其进行匹配。
2. 匹配规则：主牌区卡牌点数与手牌顶部点数差 `1` 即可匹配，花色不限。
3. 当主牌区无可匹配牌时，可点击备用牌堆翻新牌替换手牌顶部。
4. 回退规则：支持多步回退，按点击历史逆序回退，卡牌需反向 `MoveTo` 回原位置。
5. 覆盖规则：覆盖状态卡牌不可操作，需先移除其上层卡牌后才可操作。
6. 画面基础：设计分辨率 `1080x2080`，主牌区 `1080x1500`，堆牌区 `1080x580`。
7. 架构规范：MVC + `configs/models/views/controllers/managers/services/utils` 分层。
